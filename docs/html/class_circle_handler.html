◆ CircleHandler() [1/2]

CircleHandler::CircleHandler ( )

Default constructor calling other constructor with 10 circles with a radius of ten. The time of each state is 3 seconds and initial state is dark (visible, but not active jet). The circles are blue and created in a window of size 480x640.

CircleHandler.cpp

Handles timers and visualization of circles.
